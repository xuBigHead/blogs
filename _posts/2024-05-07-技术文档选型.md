# 技术文档选型

## **在线文档功能对比**

| **功能点**    | **TAPD文档**         | **企业微信文档**      | **钉钉文档**        | 飞书文档 |
|------------|--------------------|-----------------|-----------------|------|
| 富文本语法      | 支持                 | 支持              | 支持              |      |
| Markdown语法 | 支持，和富文本编辑方式隔离      | 支持，和富文本编辑方式同时支持 | 支持，和富文本编辑方式同时支持 |      |
| 目录         | 支持，嵌入到文档中，不支持展开和收起 | 支持              | 支持              |      |
| 画图         | 支持                 | 支持              | 支持              |      |
| 导出PDF、Word | 支持                 | 支持              | 支持              |      |
| 历史版本       | 支持                 | 支持              | 支持              |      |
| 编辑查看权限管理   | 支持文档级别             | 支持文件夹、文档级别      | 支持文件夹、文档级别      |      |
| 文档管理       | 支持多级文件夹            | 支持多级文件夹         | 支持多级文件夹         |      |
| 分享         | 不支持                | 支持              | 支持              |      |
| 表格文档       | 不支持                | 支持              | 支持              |      |
| 人员组成       | 所有人员               | 部分人员            | 部分人员            |      |
