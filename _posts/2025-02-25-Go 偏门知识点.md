# Go 偏门知识点

## 语法相关

##### 在某些很少见的场景中，圆括号是必需的

```go
func TestRequiredParentheses(t *testing.T) {
	// 因为{}的烦扰，下面这三行均编译失败。
	/*
		if T{} == T{123, 789} {}
		if T{} == (T{123, 789}) {}
		if (T{}) == T{123, 789} {}
		var _ = func()(nil) // nil被认为是一个类型
	*/
	
	if (T{} == T{123, 789}) {
	}
	if (T{}) == (T{123, 789}) {
	}
	var _ = (func())(nil) // nil被认为是一个值
}
```
